# 3-in-1 Smart Robot Car

## Overview

The 3-in-1 Smart Robot Car is an embedded systems project developed using Arduino Uno. It integrates three operating modes into a single robotic platform: IR Remote Control, Line Following, and Obstacle Avoidance. The robot can switch between manual and autonomous operation, demonstrating real-time sensor processing, motor control, and embedded system design.

---

## Features

- IR Remote Controlled Navigation
- Automatic Line Following
- Autonomous Obstacle Avoidance
- Mode Switching via IR Remote
- PWM-Based Motor Speed Control
- Real-Time Sensor Processing

## Working Modes

### IR Remote Control
The robot can be manually controlled using an IR remote for forward, backward, left, right, and stop movements.

### Line Following
Using IR sensors, the robot detects and follows a predefined path automatically.

### Obstacle Avoidance
The robot uses an ultrasonic sensor mounted on a servo motor to detect obstacles, scan its surroundings, and choose a clear path for navigation.
